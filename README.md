# RTopK
Official Implementation of "RTop-K: Ultra-Fast Row-Wise Top-K Selection for Neural Network Acceleration on GPUs"

## Clone and build the project
```sh
git clone https://github.com/xiexi51/RTopK.git
cd RTopK
mkdir build
cd build
cmake ..
make
```

## Run the test
```sh
./rtopk
```

## Requirements
- CUDA Toolkit (>= 12.0)
- CMake (>= 3.5)
- C++ Standard (>= 17)
